# React + Vite

Этот проект создан с использованием React 19.2.1 и Vite 7.3.0.

## Установка зависимостей

```bash
npm install
```

## Запуск dev-сервера

```bash
npm run dev
```

## Сборка для production

```bash
npm run build
```

## Просмотр production сборки

```bash
npm run preview
```
